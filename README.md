# UI Component library

This is a base project to start your own UI Component Library for React

We use

* Vite 
* Storybook
* Rollup
* Theme with Emotion
* Typescript

## Installation

Use the yarn or npm to install

```bash
npm install
```

Run storybook

```bash
npm run storybook
```
